## Что это

Геймифицированный гайд по выходу на маркетплейсы (Etsy → Amazon), стилизованный под вселенную Гарри Поттера. Один HTML-файл, работает в любом браузере, включая мобильный. Оптимизирован под экран смартфона (520px).

## Для кого

Подросток 17 лет без опыта в бизнесе и e-commerce. Читала все книги HP в оригинале. Живёт в Португалии (EU-резидент). Документ написан на русском.

## Геймификация

- **Баллы факультета** — начисляются за выполненные квесты (всего ~400+)
- **Курсы Хогвартса** — 8 уровней (Маггл → Т.Р.И.Т.О.Н.)
- **Ачивменты** — всплывающие тосты при достижении порогов
- **Прогресс** — сохраняется в localStorage браузера

## HP-факты (вшиты между разделами)

Малоизвестные факты из канона, подобранные для фаната, читавшего книги в оригинале:

- Произношение «Voldemort» без «t» (подтверждено Роулинг)
- «Dumbledore» = староанглийское «шмель»
- Фред и Джордж Уизли родились 1 апреля; Джордж не смог вызвать Патронуса после смерти Фреда
- Последняя глава «Даров Смерти» написана в 1990-м — за 7 лет до первой книги
- Рукопись отвергнута 12 издательствами
- Рэдклифф сломал ~80 палочек (использовал как барабанные)
- Цветовая символика: красный = добро, зелёный = тёмная магия
